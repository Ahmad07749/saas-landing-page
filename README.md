# FlowDesk - SaaS Product Landing Page & Customer Onboarding Portal

A fully responsive SaaS Landing Page with Customer Onboarding Portal and Dashboard Preview built using HTML, CSS, and JavaScript.

## Features
- Hero section with CTA and animations
- Features section (Bento grid layout)
- Customer testimonials
- Pricing plans with monthly/annual toggle
- FAQ accordion
- Contact form with validation
- Multi-step onboarding portal with form validation
- Interactive dashboard with sidebar navigation
- Dark/Light mode toggle
- Fully responsive (mobile, tablet, desktop)

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
